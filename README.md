Проект "Веб-склад" призван, чтобы упростить работу с БД для людей.
Реализована система регистрации и логина.
При создании склада создаётся зависимость склада от пользователя.
При добавлении товара создаётся зависимость товара от склада.
Получается следующая цепочка для вывода предметов на складе: пользователь > склад > предмет
При добавлении нельзя, чтобы цена и количество были отрицательными.
Более того, при попытке изменить название предмета, на уже существующее на складе, будет ошибка.
Для загрузки файла, необходимо создать Excel-файл, в котором первая строка обязательно
должна быть заголовками таблицы из БД (Название, Цена, Количество). Есть заранее заготовленный файл "товары.xlsx", который вы можете найти в zip-архиве проекта.
